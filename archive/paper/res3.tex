\section{Contextual Bandits}
\label{sec:exp3_results}

Experiment~3 replaces Q-learning with contextual bandits. LinUCB uses optimism-based exploration while Contextual Thompson Sampling uses posterior sampling, creating a natural contrast for how exploration style affects outcomes. Each receives a tailored factorial design to avoid wasting design cells on factors with no effect in the other algorithm.

\subsection{LinUCB}

Experiment~IIIa deploys LinUCB agents under the affiliated valuation model. The design is a $3 \times 2^7 = 384$ mixed-level factorial with 9 factors (7 binary plus the three-level affiliation parameter $\eta$), including the regularisation parameter $\lambda$ and memory decay, both of which strongly affect LinUCB performance. The design is replicated twice for \ExpThreeANobs{} observations.

\subsection{Summary}

Replacing Q-learning with contextual bandits does not improve seller welfare. Richer algorithm design does not translate into higher revenue for the auctioneer. LinUCB and Thompson Sampling respond to different design levers, validating the decision to analyse them separately. Higher exploration intensity reduces revenue under both algorithms, but the magnitudes and moderating factors differ. Global sensitivity analysis confirms these hierarchies. Under LinUCB, the number of bidders accounts for over half of revenue variance ($S_T = 0.55$) and nearly all winner entropy variance ($S_T = 0.75$). Under Thompson Sampling, factor importance is more diffuse, with no single factor exceeding $S_T = 0.31$ for revenue. Across both algorithms, tuning parameters are consistently negligible ($S_T < 0.04$ for revenue), reinforcing the conclusion that market structure dominates algorithm configuration (Appendix~\ref{sec:appendix_sensitivity}). Experiments~1--3 share a common feature, namely that agents bid directly from value estimates without external constraints. Section~\ref{sec:exp4_results} tests whether the patterns documented above persist when agents face budget constraints, the dominant paradigm in modern advertising exchanges.
